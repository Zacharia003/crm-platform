# CRM Platform

This project is a lightweight CRM designed for small businesses to
manage enquiries, orders, invoices, and payments in a single system.

## Key Features (Phase 1)

- Email-based enquiry tracking
- Lead-to-order conversion
- Purchase and sales management
- Invoice & proforma PDF generation
- Order status and payment tracking

## Tech Stack

- Backend: Spring Boot
- Frontend: React
- Database: MySQL
- Authentication: JWT
